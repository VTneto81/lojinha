# 🧸 PetitShop — Lojinha da Mamãe

Plataforma de e-commerce colaborativa para mães empreendedoras.

## Stack
- React 18 + Vite
- Tailwind CSS + shadcn/ui
- Base44 (backend as a service)
- Lucide React (icons)
- Framer Motion

## Páginas
- `/` — Loja principal
- `/produto/:id` — Detalhe do produto
- `/checkout` — Finalização de compra
- `/pedido-enviado` — Confirmação de pedido
- `/enviar-anuncio` — Cadastro de produto por parceiras
- `/parceira/:instagram` — Perfil público da parceira
- `/portal-parceira` — Área restrita da parceira
- `/admin` — Painel administrativo

## Funcionalidades
- Catálogo de produtos com filtros por categoria e parceira
- Carrinho de compras com localStorage
- Checkout via WhatsApp
- Portal da parceira (anúncios, repasses, perfil)
- Painel admin completo (produtos, pedidos, repasses, financeiro)
- Geração de legendas com IA
- Notificações automáticas via WhatsApp
- Relatórios e exportação CSV
- Comissões automáticas por parceira
